# 🏎️ Blog Formule 1 – BTS CCF

## 📖 Description
Ce projet est un **blog consacré à la Formule 1**, réalisé dans le cadre d’un **CCF de BTS**.  
Le site permet de présenter l’actualité, les pilotes, les écuries et les courses de Formule 1.

L’objectif du projet est de mettre en pratique :
- le développement web
- l’organisation d’un projet avec GitHub
- la création d’un site informatif

---

## 🚀 Fonctionnalités
- 📰 Articles sur la Formule 1  
- 👨‍🏎️ Présentation des pilotes  
- 🏁 Présentation des écuries  
- 📅 Informations sur les courses  
- 🌐 Navigation simple et responsive

---

## 🛠️ Technologies utilisées
- **HTML**
- **CSS**
- **Git / GitHub**
